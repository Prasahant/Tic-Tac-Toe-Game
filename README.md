# 🎮 Tic Tac Toe Game

This project is built to practice core JavaScript concepts like DOM manipulation, event handling, and game logic.

## 🚀 Live Demo
https://enchanting-boba-72a769.netlify.app/

## ✨ Features
- Two-player gameplay (X and O)
- Winner detection logic
- Draw condition handling
- Reset and New Game functionality
- Dynamic UI updates

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📚 Learning Focus
- DOM manipulation
- Event listeners
- Game logic implementation
- User interaction handling

- Add AI (single-player mode)
